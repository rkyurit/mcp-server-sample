# デザインシステム用コンポーネント一覧

- [x] **McpInput** - テキスト入力フィールド（既存）
- [x] **McpTextarea** - 複数行テキスト入力
- [x] **McpCheckbox** - チェックボックス
- [x] **McpRadio** - ラジオボタン
